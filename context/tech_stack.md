# Tech Stack

## Frontend

* React
* TypeScript
* Vite
* Tailwind CSS
* Framer Motion

## Backend

* Node.js
* Express.js

## Database

* PostgreSQL

## ORM

* Prisma

## Authentication

* JWT Authentication

## File Storage

### Development

* Local Storage

### Production

* Supabase Storage or Amazon S3

| Technology         | Purpose                         |
| ------------------ | ------------------------------- |

| Framer Motion      | Animations and card transitions |
| Node.js            | Backend runtime                 |
| Express.js         | API development                 |
| PostgreSQL         | Data storage                    |
| Prisma             | Database access and management  |
| JWT                | User authentication             |
| Local Storage / S3 | Document storage                |
|                    |                                 |
